2. 注册
3. 首页
4. 社区公告
5. 维修工
6. 维修订单、接单、交流
7. 新闻资讯
8. 留言反馈

10. 发布维修
11. 我的接单信息
12. 我的订单信息
13. 在线沟通
14. 我的举报信息

**维修工功能：**

3. 首页

8. 留言反馈
